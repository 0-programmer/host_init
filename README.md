# 🚀 Ubuntu Setup Script

Этот скрипт устанавливает базовый набор инструментов разработчика на "чистую" Ubuntu:

- 🐍 Python 3 + pip
- 🐙 Git
- 🐟 Fish Shell
- 🐳 Docker + Docker Compose
- 🟢 Node.js (LTS) + npm
- 🛠️ Полезные утилиты: curl, wget, unzip, vim, htop, build-essential и др.
